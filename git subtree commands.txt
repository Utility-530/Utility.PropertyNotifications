https://opensource.com/article/20/5/git-submodules-subtrees

-- add remote
- this adds the folder to the master folder structure (ensure folder doesn't already exist)

$ git remote add <remote-name> <URL to Git repo>

example
git remote add Structs https://github.com/Utility-530/Utility.Structs.git

-- add repo as subtree
this ensures the changes in the subfolder get noticed

$ git subtree add --prefix <folder> <URL to Git repo> <subtree-branchname> --squash?	

example
git subtree add --prefix Structs https://github.com/Utility-530/Utility.Structs.git master*

ensure branch names match
